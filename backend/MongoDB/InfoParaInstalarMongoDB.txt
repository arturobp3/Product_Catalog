Para importar debe estar el servidor puesto

Descargar MongoDB Server
------------------------
https://www.mongodb.com/dr/fastdl.mongodb.org/win32/mongodb-win32-x86_64-2008plus-ssl-4.0.9.zip/download

Descargar Robo 3T (Opcional), y asegurarse de que ambos puertos (tanto el de MongoDB Server y Robo 3T) son el 27017
---------------------------------
https://robomongo.org/download

Instalar driver MongoDB para PHP
---------------------------------
Para ello mirar la versión de tu PHP, la arquitectura y el Thread Safe (utilizar el fichero infoPHP.php
contenido 
y buscar cual se corresponde con el tuyo.
Descargar el driver del siguiente link:
https://pecl.php.net/package/mongodb/1.5.3/windows

Copiamos el archivo php_mongodb.dll y lo pegamos en la carpeta xampp/php/ext
En la carpeta xampp/php editamos el archivo php.ini y copiamos la siguiente sentencia:
	extension=php_mongodb.dll
Comprobamos en la pagina infoPHP.php que se ha instalado correctamente mongoDB



- Ejecutar mongod.exe. Se encuentra en la carpeta 'bin' de MongoDB Server

- Abrir la consola del sistema (CMD), navegar hasta la carpeta 'bin' de MongoDB Server y ejecutar
el siguiente comando: 
	mongoimport -h localhost:27017 --db Product_Catalogue --collection InfoProducto --file C:\xampp\htdocs\
	Product_Catalogue\backend\MongoDB\databaseMongoDB.json

	mongoexport --db Product_Catalogue --collection InfoProducts --out C:\rutaquequieras
